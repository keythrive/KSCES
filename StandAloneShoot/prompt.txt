Microsoft デザイナ、個人アカウントでログインしなおして、AIで作成、画像を作る

# Background.png
シューティングゲームの背景用画像で、深い黒い闇を背景としつつ、渦巻状の星雲が散らばっている銀河団をモチーフにした、宇宙空間のアニメ風イラスト

# Fighter.png
シューティングゲームで自分の操縦する1機の戦闘機のピクセルアート。色はダークシルバーで向きは下向き、ミサイルとレーザ砲を具備して、背景は無色。

# Ufo.png
宇宙エイリアンが操縦する1機の攻撃型UFOのピクセルアート。色はマゼンタとダークグリーンを基調としてして、背景は無色。
上向きで攻撃的な形を特徴とする。

# Beam.png
戦闘機が発射する下向きに飛ぶ1発のレーザビーム砲で、色はオレンジを基調として背景は無色のピクセルアート。

# Missile.png
上向きに飛ぶ敵のミサイルで、色は濃い黄色を基調として背景は無色のピクセルアート。

# Broken.png
破壊されて消滅する視覚的効果を表すピクセルアートで背景は透明

-----

パッケージとしてpygameを使って、縦スクロールで下から上にスクロールするシューティングゲームのPythonコードを生成する。
詳しい仕様は次のとおり：
# 使用する画像ファイル
・画面の背景画像として、background.pngを使う
・自分の戦闘機画像として、fighter.pngを使う
・敵のUFOとして、ufo.pngを使う
・自分が発射するレーザビームとして、Beam.pngを使う
・敵が発射するミサイルとして、Missile.pngを使う
・敵のミサイルが自分の戦闘機に衝突した判定がTrueの時、Broken.pngを使う
・画面の左下に、現在のスコア（初期値＝00000）を表示する

＃自分の戦闘機の操縦
・カーソルキー、上下左右を使って移動する
・スペースキーを使うと、下向きにレーザビームを発射する
・自分の戦闘機が、敵のUFOや敵のミサイルに衝突判定が成立したらゲームオーバーとする

#敵のUFOの動き方
・一定時間ごとに、敵のUFOを画面下端から発生させて、画面上端に来るまで、上方向に移動する
・敵UFOの動き方は、三角関数sin,cosのような波の振幅をもって揺れて動く
・画面の外に出たら敵は消滅して消える
・ランダムな時間間隔で、敵のUFOは自分の戦闘機に向けてミサイルを連続発射される

# 自戦闘機が発射するレーザ光線の動作
・下向きに移動させて、敵UFOにあたるか、下端に到達したら消滅する
・敵UFOにあたった場合は、敵UFOとレーザ光線ともに消滅させて、しばらくBroken.pngを表示した後、スコアを100点加算する
